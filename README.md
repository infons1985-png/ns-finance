# N&S Finance

Private business app for collecting invoices, uploading physical receipts, loading bank statements, and sorting documents by payment month.

## Current version

- Premium dark/gold dashboard
- Invoice inbox
- PDF/image upload preview
- Physical receipt upload flow
- Bank statement upload placeholder
- Demo matching logic for July 2026
- Supabase client configured
- PWA manifest
